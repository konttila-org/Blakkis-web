# Blakkis-web 🐐

Welcome to the Blakkis-web repository! This project is built using Node.js (v20+), npm (v10+), and features a frontend vite application.

## Getting Started 🛠️

### 1. Install Dependencies 📦

Install the necessary dependencies by running:

```bash
npm install
```

### 2. Start the Website

Launch the Vite application:

```bash
npm run dev
```
